# OlympusGCP-Messaging

High-intent cluster for GCP event-driven and messaging services (Pub/Sub, Eventarc).

## Domain: 32000 (Message Hubs)

This workspace provides:
1.  **Local Emulators**: Pre-configured Pub/Sub and event bus emulators.
2.  **MCP Bridges**: Go-based servers for event publishing and subscription management.
